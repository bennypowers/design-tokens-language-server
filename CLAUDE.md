## About

This is a thin wrapper around [asimonim](https://github.com/bennypowers/asimonim).
All LSP logic lives in the asimonim repo. This repo exists only to build the
`design-tokens-language-server` binary for backward compatibility.

## Go

Use go 1.25+ features. Use `go vet`.

## Git

When commit messages mention AI agents, always use `Assisted-By`, never use `Co-Authored-By`.
